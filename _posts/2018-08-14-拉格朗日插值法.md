人话概括:  

已知$f(x)$上n个点$(x_i,y_i)$,求$f(k)$  
对于$(x_i,y_i)$找到L(x)使得$L(x_i)==y_i$,$L(x_j)==0$  
然后你就可以认为$f(K)=\sum L(K)$

		<h1 class = "postTitle">
			<a id="cb_post_title_url" class="postTitle2" href="https://www.cnblogs.com/ECJTUACM-873284962/p/6833391.html">拉格朗日插值法(图文详解)</a>
		</h1>
		<div class="clear"></div>
		<div class="postBody">
			<div id="cnblogs_post_body" class="blogpost-body"><p>在<a title="数值分析" href="http://zh.wikipedia.org/wiki/%E6%95%B0%E5%80%BC%E5%88%86%E6%9E%90" rel="nofollow">数值分析</a>中，<strong>拉格朗日插值法</strong>是以<a title="法国" href="http://zh.wikipedia.org/wiki/%E6%B3%95%E5%9B%BD" rel="nofollow">法国</a>十八世纪<a title="数学家" href="http://zh.wikipedia.org/wiki/%E6%95%B0%E5%AD%A6%E5%AE%B6" rel="nofollow">数学家</a><a title="约瑟夫&middot;拉格朗日" href="http://zh.wikipedia.org/wiki/%E7%BA%A6%E7%91%9F%E5%A4%AB%C2%B7%E6%8B%89%E6%A0%BC%E6%9C%97%E6%97%A5" rel="nofollow">约瑟夫&middot;拉格朗日</a>命名的一种<a title="多项式插值" href="http://zh.wikipedia.org/wiki/%E5%A4%9A%E9%A1%B9%E5%BC%8F%E6%8F%92%E5%80%BC" rel="nofollow">多项式插值</a>方法。许多实际问题中都用<a title="函数" href="http://zh.wikipedia.org/wiki/%E5%87%BD%E6%95%B0" rel="nofollow">函数</a>来表示某种内在联系或规律，而不少函数都只能通过实验和观测来了解。如对实践中的某个<a class="mw-redirect" title="物理" href="http://zh.wikipedia.org/wiki/%E7%89%A9%E7%90%86" rel="nofollow">物理</a>量进行观测，在若干个不同的地方得到相应的观测值，拉格朗日插值法可以找到一个<a class="mw-redirect" title="多项式" href="http://zh.wikipedia.org/wiki/%E5%A4%9A%E9%A1%B9%E5%BC%8F" rel="nofollow">多项式</a>，其恰好在各个观测的点取到观测到的值。这样的多项式称为<strong>拉格朗日（插值）多项式</strong>。<a title="数学" href="http://zh.wikipedia.org/wiki/%E6%95%B0%E5%AD%A6" rel="nofollow">数学</a>上来说，拉格朗日插值法可以给出一个恰好穿过二维<a title="平面" href="http://zh.wikipedia.org/wiki/%E5%B9%B3%E9%9D%A2" rel="nofollow">平面</a>上若干个已知点的多项式函数。拉格朗日插值法最早被<a title="英国" href="http://zh.wikipedia.org/wiki/%E8%8B%B1%E5%9B%BD" rel="nofollow">英国</a>数学家<a title="爱德华&middot;华林" href="http://zh.wikipedia.org/wiki/%E7%88%B1%E5%BE%B7%E5%8D%8E%C2%B7%E5%8D%8E%E6%9E%97" rel="nofollow">爱德华&middot;华林</a>于1779年发现<sup id="cite_ref-1"><a href="http://zh.wikipedia.org/wiki/%E6%8B%89%E6%A0%BC%E6%9C%97%E6%97%A5%E6%8F%92%E5%80%BC%E6%B3%95#cite_note-1" rel="nofollow">[1]</a></sup>，不久后（1783年）由<a class="mw-redirect" title="莱昂哈德&middot;欧拉" href="http://zh.wikipedia.org/wiki/%E8%8E%B1%E6%98%82%E5%93%88%E5%BE%B7%C2%B7%E6%AC%A7%E6%8B%89" rel="nofollow">莱昂哈德&middot;欧拉</a>再次发现。1795年，拉格朗日在其著作《师范学校数学基础教程》中发表了这个插值方法，从此他的名字就和这个方法联系在一起<sup id="cite_ref-2"><a href="http://zh.wikipedia.org/wiki/%E6%8B%89%E6%A0%BC%E6%9C%97%E6%97%A5%E6%8F%92%E5%80%BC%E6%B3%95#cite_note-2" rel="nofollow">[2]</a></sup>。</p>
<p>对于给定的若<strong>n+1</strong>个点<img src="http://upload.wikimedia.org/math/e/d/1/ed1214922146517590674f2530b8c1d0.png" alt="(x_{0},y_{0}),(x_{1},y_{1}),\ldots ,(x_{n},y_{n})" />，对应于它们的次数不超过<strong>n</strong>的拉格朗日多项式<img src="http://upload.wikimedia.org/math/b/1/d/b1dc21bfe298d9c5408dced55789670b.png" alt="\scriptstyle L" />只有一个。如果计入次数更高的多项式，则有无穷个，因为所有与<img src="http://upload.wikimedia.org/math/b/1/d/b1dc21bfe298d9c5408dced55789670b.png" alt="\scriptstyle L" />相差<img src="http://upload.wikimedia.org/math/e/7/9/e7934de5e4a3545f0947388ba10573b4.png" alt="\lambda (x-x_{0})(x-x_{1})\ldots (x-x_{n})" />的多项式都满足条件。例子：</p>
<p><a class="image" href="http://zh.wikipedia.org/wiki/File:Lagrange_polynomial.svg" rel="nofollow"><img class="thumbimage" src="http://upload.wikimedia.org/wikipedia/commons/thumb/5/5a/Lagrange_polynomial.svg/220px-Lagrange_polynomial.svg.png" alt="" width="220" height="219" /></a></p>
<div>已知平面上四个点：<span style="color: #b30000;">(-9, 5)</span>,<span style="color: #0000b3;">(-4, 2)</span>,&nbsp;<span style="color: #00b300;">(-1, -2)</span>,&nbsp;<span style="color: #b3b300;">(7, 9)</span>，拉格朗日多项式：<span style="color: #333333;"><em>L</em>（<em>x</em>）</span>（黑色）穿过所有点。而每个基本多项式：<img src="https://images2017.cnblogs.com/blog/1100338/201710/1100338-20171019184215552-1972274693.png" alt="" />
<p>&nbsp;</p>
以及<span style="color: #b3b300;"><span style="color: #b3b300;"><img src="https://images2017.cnblogs.com/blog/1100338/201710/1100338-20171019184237584-1303677761.png" alt="" /></span></span>各穿过对应的一点，并在其它的三个点的<em>x</em>值上取零。</div>
<div>
<h2><span id=".E5.AE.9A.E4.B9.89">定义</span></h2>
<div>
<p>对某个多项式<a title="函数" href="http://zh.wikipedia.org/wiki/%E5%87%BD%E6%95%B0" rel="nofollow">函数</a>，已知有给定的<em>k</em>&nbsp;+&nbsp;1个取值点：</p>
<dl><dd><img src="http://upload.wikimedia.org/math/d/8/a/d8ab7592d996df5c3d84c390776c1bc0.png" alt="(x_{0},y_{0}),\ldots ,(x_{k},y_{k})" /></dd></dl>
<p>其中<img src="http://upload.wikimedia.org/math/7/7/d/77dc7a56686018973d465a321f70b9a5.png" alt="x_{j}" />对应着<a class="new" title="自变量（页面不存在）" href="http://zh.wikipedia.org/w/index.php?title=%E8%87%AA%E5%8F%98%E9%87%8F&amp;action=edit&amp;redlink=1" rel="nofollow">自变量</a>的位置，而<img src="http://upload.wikimedia.org/math/8/2/b/82be2cc41eff2674791117bef7a05a88.png" alt="y_{j}" />对应着函数在这个位置的取值。</p>
<p>假设任意两个不同的<em>x</em><sub><em>j</em></sub>都互不相同，那么应用拉格朗日插值公式所得到的<strong>拉格朗日插值多项式</strong>为：</p>
<dl><dd><img src="http://upload.wikimedia.org/math/7/1/c/71c9d5496cd18131bf73d1402477c003.png" alt="L(x):=\sum _{{j=0}}^{{k}}y_{j}\ell _{j}(x)" /></dd></dl>
<p>其中每个<img src="http://upload.wikimedia.org/math/7/0/2/702fdcdcfa7539950089909d85e12263.png" alt="\ell _{j}(x)" />为<strong>拉格朗日基本多项式</strong>（或称<strong>插值基函数</strong>），其表达式为：</p>
<dl><dd><img src="http://upload.wikimedia.org/math/9/9/a/99a8f9408d532d4713380f898affe049.png" alt="\ell _{j}(x):=\prod _{{i=0,\,i\neq j}}^{{k}}{\frac  {x-x_{i}}{x_{j}-x_{i}}}={\frac  {(x-x_{0})}{(x_{j}-x_{0})}}\cdots {\frac  {(x-x_{{j-1}})}{(x_{j}-x_{{j-1}})}}{\frac  {(x-x_{{j+1}})}{(x_{j}-x_{{j+1}})}}\cdots {\frac  {(x-x_{{k}})}{(x_{j}-x_{{k}})}}." /><sup id="cite_ref-3"><a href="http://zh.wikipedia.org/wiki/%E6%8B%89%E6%A0%BC%E6%9C%97%E6%97%A5%E6%8F%92%E5%80%BC%E6%B3%95#cite_note-3" rel="nofollow">[3]</a></sup></dd></dl>
<p>拉格朗日基本多项式<img src="http://upload.wikimedia.org/math/7/0/2/702fdcdcfa7539950089909d85e12263.png" alt="\ell _{j}(x)" />的特点是在<img src="http://upload.wikimedia.org/math/7/7/d/77dc7a56686018973d465a321f70b9a5.png" alt="x_{j}" />上取值为<strong>1</strong>，在其它的点<img src="http://upload.wikimedia.org/math/6/d/6/6d6b634c2230f4b58638271fc4a26133.png" alt="x_{i},\,i\neq j" />上取值为<strong>0</strong>。</p>
<h2><span id=".E7.AF.84.E4.BE.8B">范例</span></h2>
<p>假设有某个二次多项式函数<img src="http://upload.wikimedia.org/math/8/f/a/8fa14cdd754f91cc6554c9e71929cce7.png" alt="f" />，已知它在三个点上的取值为：</p>
<ul>
<li><img src="http://upload.wikimedia.org/math/5/6/6/5667d782a84f3cec6834d5b6d2492b88.png" alt="f(4)=10" /></li>
<li><img src="http://upload.wikimedia.org/math/b/a/9/ba9c61595aefe8c9786504b9acf10e96.png" alt="f(5)=5.25" /></li>
<li><img src="http://upload.wikimedia.org/math/4/c/e/4ce6f08d96e6c63c0745346da7f6f4d7.png" alt="f(6)=1" /></li>
</ul>
<p>要求<img src="http://upload.wikimedia.org/math/b/e/7/be7fd678d34cea40215e71e398dadb01.png" alt="f(18)" />的值。</p>
<p>首先写出每个拉格朗日基本多项式：</p>
<dl><dd><img src="http://upload.wikimedia.org/math/0/6/a/06a8bcd0f2017e3f3cff9016d25fa120.png" alt="\ell _{0}(x)={\frac  {(x-5)(x-6)}{(4-5)(4-6)}}" /></dd><dd><img src="http://upload.wikimedia.org/math/1/0/c/10c86451a0e24d30237a5a9c54ea4709.png" alt="\ell _{1}(x)={\frac  {(x-4)(x-6)}{(5-4)(5-6)}}" /></dd><dd><img src="http://upload.wikimedia.org/math/1/f/c/1fc113255cf9ba7dd5a0abc7e0a586ad.png" alt="\ell _{2}(x)={\frac  {(x-4)(x-5)}{(6-4)(6-5)}}" /></dd></dl>
<p>然后应用拉格朗日插值法，就可以得到<img src="http://upload.wikimedia.org/math/8/3/8/83878c91171338902e0fe0fb97a8c47a.png" alt="p" />的表达式（<img src="http://upload.wikimedia.org/math/8/3/8/83878c91171338902e0fe0fb97a8c47a.png" alt="p" />为函数<img src="http://upload.wikimedia.org/math/8/f/a/8fa14cdd754f91cc6554c9e71929cce7.png" alt="f" />的插值函数）：</p>
<dl><dd><img src="http://upload.wikimedia.org/math/1/8/5/18589b8dcf40c99ef9348960a97c849a.png" alt="p(x)=f(4)\ell _{0}(x)+f(5)\ell _{1}(x)+f(6)\ell _{2}(x)" /></dd><dd><img src="http://upload.wikimedia.org/math/7/3/b/73b2c4750437572e6da99724b9add251.png" alt=".\,\,\,\,\,\,\,\,\,\,=10\cdot {\frac  {(x-5)(x-6)}{(4-5)(4-6)}}+5.25\cdot {\frac  {(x-4)(x-6)}{(5-4)(5-6)}}+1\cdot {\frac  {(x-4)(x-5)}{(6-4)(6-5)}}" /><br /><img src="http://upload.wikimedia.org/math/4/d/d/4dd811dcba2fd7c558a9141a0d9c6d60.png" alt=".\,\,\,\,\,\,\,\,\,\,={\frac  {1}{4}}(x^{2}-28x+136)" /></dd></dl>
<p>此时代入数值<img src="http://upload.wikimedia.org/math/a/7/b/a7bd0c046451b540e2fb253a519f8fe2.png" alt="\ 18" />就可以求出所需之值：<img src="http://upload.wikimedia.org/math/7/2/7/7272a5f44c7a3cf0576e5c93e198a311.png" alt="\ f(18)=p(18)=-11" />。</p>
<h2><span id=".E8.AF.81.E6.98.8E">证明</span></h2>
<h3><span id=".E5.AD.98.E5.9C.A8.E6.80.A7">存在性</span></h3>
<p>对于给定的<em>k</em>+1个点：<img src="http://upload.wikimedia.org/math/d/8/a/d8ab7592d996df5c3d84c390776c1bc0.png" alt="(x_{0},y_{0}),\ldots ,(x_{k},y_{k})" />，拉格朗日插值法的思路是找到一个在一点<img src="http://upload.wikimedia.org/math/7/7/d/77dc7a56686018973d465a321f70b9a5.png" alt="x_{j}" />取值为<strong>1</strong>，而在其他点取值都是<strong>0</strong>的多项式<img src="http://upload.wikimedia.org/math/7/0/2/702fdcdcfa7539950089909d85e12263.png" alt="\ell _{j}(x)" />。这样，多项式<img src="http://upload.wikimedia.org/math/1/3/8/138ce67309ae8a4ac65c5bf83599189c.png" alt="y_{j}\ell _{j}(x)" />在点<img src="http://upload.wikimedia.org/math/7/7/d/77dc7a56686018973d465a321f70b9a5.png" alt="x_{j}" />取值为<img src="http://upload.wikimedia.org/math/8/2/b/82be2cc41eff2674791117bef7a05a88.png" alt="y_{j}" />，而在其他点取值都是<strong>0</strong>。而多项式<img src="http://upload.wikimedia.org/math/7/1/c/71c9d5496cd18131bf73d1402477c003.png" alt="L(x):=\sum _{{j=0}}^{{k}}y_{j}\ell _{j}(x)" />就可以满足</p>
<dl><dd><img src="http://upload.wikimedia.org/math/8/a/7/8a79edd796958696834695588887dd39.png" alt="L(x_{j})=\sum _{{i=0}}^{{k}}y_{i}\ell _{i}(x_{j})=0+0+\cdots +y_{j}+\cdots +0=y_{j}" /></dd></dl>
<p>在其它点取值为<strong>0</strong>的多项式容易找到，例如：</p>
<dl><dd><img src="http://upload.wikimedia.org/math/0/d/7/0d7b2c089f7d8374dc1da7b2a3f0e772.png" alt="(x-x_{0})\cdots (x-x_{{j-1}})(x-x_{{j+1}})\cdots (x-x_{{k}})" /></dd></dl>
<p>它在点<img src="http://upload.wikimedia.org/math/7/7/d/77dc7a56686018973d465a321f70b9a5.png" alt="x_{j}" />取值为：<img src="http://upload.wikimedia.org/math/9/d/3/9d3042dc49e5d32a15fc40c9651295fa.png" alt="(x_{j}-x_{0})\cdots (x_{j}-x_{{j-1}})(x_{j}-x_{{j+1}})\cdots (x_{j}-x_{{k}})" />。由于已经假定<img src="http://upload.wikimedia.org/math/6/4/b/64b94619c0031522d16b05a37bc06189.png" alt="x_{i}" />两两互不相同，因此上面的取值不等于<strong>0</strong>。于是，将多项式除以这个取值，就得到一个满足&ldquo;在<img src="http://upload.wikimedia.org/math/7/7/d/77dc7a56686018973d465a321f70b9a5.png" alt="x_{j}" />取值为<strong>1</strong>，而在其他点取值都是<strong>0</strong>的多项式&rdquo;：</p>
<dl><dd><img src="http://upload.wikimedia.org/math/d/6/2/d62a2ffab8cf2363fcc1f26f388838d3.png" alt="\ell _{j}(x):=\prod _{{i=0,\,i\neq j}}^{{k}}{\frac  {x-x_{i}}{x_{j}-x_{i}}}={\frac  {(x-x_{0})}{(x_{j}-x_{0})}}\cdots {\frac  {(x-x_{{j-1}})}{(x_{j}-x_{{j-1}})}}{\frac  {(x-x_{{j+1}})}{(x_{j}-x_{{j+1}})}}\cdots {\frac  {(x-x_{{k}})}{(x_{j}-x_{{k}})}}" /></dd></dl>
<p>这就是拉格朗日基本多项式。</p>
<h3><span id=".E5.94.AF.E4.B8.80.E6.80.A7">唯一性</span></h3>
<p>次数不超过<em>k</em>的拉格朗日多项式至多只有一个，因为对任意两个次数不超过<em>k</em>的拉格朗日多项式：<img src="http://upload.wikimedia.org/math/0/1/9/019b577dc46e4eaa56b825ab90fce65f.png" alt="P_{1}" />和<img src="http://upload.wikimedia.org/math/5/f/5/5f57ab9ed57ef716654939ff1c5ff233.png" alt="P_{2}" />，它们的差<img src="http://upload.wikimedia.org/math/f/0/f/f0fd7d15091b76ed04d26fd0cebb4797.png" alt="P_{1}-P_{2}" />在所有<em>k</em>+1个点上取值都是<strong>0</strong>，因此必然是多项式<img src="http://upload.wikimedia.org/math/5/b/b/5bb354c3c7668b7ab7ccf22342ca2b7e.png" alt="(x-x_{0})(x-x_{{1}})\cdots (x-x_{{k}})" />的倍数。因此，如果这个差<img src="http://upload.wikimedia.org/math/f/0/f/f0fd7d15091b76ed04d26fd0cebb4797.png" alt="P_{1}-P_{2}" />不等于<strong>0</strong>，次数就一定不小于<em>k</em>+1。但是<img src="http://upload.wikimedia.org/math/f/0/f/f0fd7d15091b76ed04d26fd0cebb4797.png" alt="P_{1}-P_{2}" />是两个次数不超过<em>k</em>的多项式之差，它的次数也不超过<em>k</em>。所以<img src="http://upload.wikimedia.org/math/8/4/a/84a74713390a39e031541924e98bb511.png" alt="P_{1}-P_{2}=0" />，也就是说<img src="http://upload.wikimedia.org/math/8/5/b/85bddd57e801457e4add1b8557f1ca52.png" alt="P_{1}=P_{2}" />。这样就证明了唯一性<sup id="cite_ref-4"><a href="http://zh.wikipedia.org/wiki/%E6%8B%89%E6%A0%BC%E6%9C%97%E6%97%A5%E6%8F%92%E5%80%BC%E6%B3%95#cite_note-4" rel="nofollow">[4]</a></sup>。</p>
<h2><span id=".E5.87.A0.E4.BD.95.E6.80.A7.E8.B4.A8">几何性质</span></h2>
<p>拉格朗日插值法中用到的拉格朗日基本多项式<img src="http://upload.wikimedia.org/math/2/e/b/2eb78febc40072965d65008f52c04a44.png" alt="\ell _{0},\ell _{1},\cdots ,\ell _{n}" />（由某一组<img src="http://upload.wikimedia.org/math/3/6/b/36b760fcdf5e19336faf4d82d5b30813.png" alt="x_{0}&lt;x_{1}&lt;\cdots &lt;x_{n}" />确定）可以看做是由次数不超过<strong>n</strong>的多项式所组成的<a class="mw-redirect" title="线性空间" href="http://zh.wikipedia.org/wiki/%E7%BA%BF%E6%80%A7%E7%A9%BA%E9%97%B4" rel="nofollow">线性空间</a>：<img src="http://upload.wikimedia.org/math/9/4/a/94a1ed21235d59c5b5f5960c561d41a3.png" alt="{\mathbb  {K}}_{n}[X]" />的一组<a class="mw-redirect" title="基底" href="http://zh.wikipedia.org/wiki/%E5%9F%BA%E5%BA%95" rel="nofollow">基底</a>。首先，如果存在一组<a title="系数" href="http://zh.wikipedia.org/wiki/%E7%B3%BB%E6%95%B0" rel="nofollow">系数</a>：<img src="http://upload.wikimedia.org/math/d/0/c/d0c4933c1514255774bd6cef9517f899.png" alt="\lambda _{0},\lambda _{1},\cdots ,\lambda _{n}" />使得，</p>
<dl><dd><img src="http://upload.wikimedia.org/math/e/3/8/e38f19217a2155d7587f97eb21d6713a.png" alt="P=\lambda _{0}\ell _{0}+\lambda _{1}\ell _{1}+\cdots +\lambda _{n}\ell _{n}=0" />，</dd></dl>
<p>那么，一方面多项式<strong>P</strong>是满足<img src="http://upload.wikimedia.org/math/9/4/c/94cda012bc78105f11a94f91c305d963.png" alt="P(x_{0})=\lambda _{0},P(x_{1})=\lambda _{1},\cdots ,P(x_{n})=\lambda _{n}" />的拉格朗日插值多项式，另一方面<strong>P</strong>是零多项式，所以取值永远是<strong>0</strong>。所以</p>
<dl><dd><img src="http://upload.wikimedia.org/math/1/9/9/199cfa746ac025b218e53c063a2a4e48.png" alt="\lambda _{0}=\lambda _{1}=\cdots =\lambda _{n}=0" />。</dd></dl>
<p>这证明了<img src="http://upload.wikimedia.org/math/2/e/b/2eb78febc40072965d65008f52c04a44.png" alt="\ell _{0},\ell _{1},\cdots ,\ell _{n}" />是<a class="mw-redirect" title="线性无关" href="http://zh.wikipedia.org/wiki/%E7%BA%BF%E6%80%A7%E6%97%A0%E5%85%B3" rel="nofollow">线性无关</a>的。同时它一共包含<strong>n+1</strong>个多项式，恰好等于<img src="http://upload.wikimedia.org/math/9/4/a/94a1ed21235d59c5b5f5960c561d41a3.png" alt="{\mathbb  {K}}_{n}[X]" />的维数。所以<img src="http://upload.wikimedia.org/math/2/e/b/2eb78febc40072965d65008f52c04a44.png" alt="\ell _{0},\ell _{1},\cdots ,\ell _{n}" />构成了<img src="http://upload.wikimedia.org/math/9/4/a/94a1ed21235d59c5b5f5960c561d41a3.png" alt="{\mathbb  {K}}_{n}[X]" />的一组基底。</p>
<p>拉格朗日基本多项式作为基底的好处是所有的多项式都是齐次的（都是<strong>n</strong>次多项式）。</p>
<h2><span id=".E4.BC.98.E7.82.B9.E4.B8.8E.E7.BC.BA.E7.82.B9">优点与缺点</span></h2>
<p>拉格朗日插值法的公式结构整齐紧凑，在理论分析中十分方便，然而在计算中，当插值点增加或减少一个时，所对应的基本多项式就需要全部重新计算，于是整个公式都会变化，非常繁琐<sup id="cite_ref-5"><a href="http://zh.wikipedia.org/wiki/%E6%8B%89%E6%A0%BC%E6%9C%97%E6%97%A5%E6%8F%92%E5%80%BC%E6%B3%95#cite_note-5" rel="nofollow">[5]</a></sup>。这时可以用重心拉格朗日插值法或<a class="mw-redirect" title="牛顿插值法" href="http://zh.wikipedia.org/wiki/%E7%89%9B%E9%A0%93%E6%8F%92%E5%80%BC%E6%B3%95" rel="nofollow">牛顿插值法</a>来代替。此外，当插值点比较多的时候，拉格朗日插值多项式的次数可能会很高，因此具有<a title="数值稳定性" href="http://zh.wikipedia.org/wiki/%E6%95%B0%E5%80%BC%E7%A8%B3%E5%AE%9A%E6%80%A7" rel="nofollow">数值不稳定</a>的特点，也就是说尽管在已知的几个点取到给定的数值，但在附近却会和&ldquo;实际上&rdquo;的值之间有很大的偏差（如右下图）<sup id="cite_ref-6"><a href="http://zh.wikipedia.org/wiki/%E6%8B%89%E6%A0%BC%E6%9C%97%E6%97%A5%E6%8F%92%E5%80%BC%E6%B3%95#cite_note-6" rel="nofollow">[6]</a></sup>。这类现象也被称为<a title="龙格现象" href="http://zh.wikipedia.org/wiki/%E9%BE%99%E6%A0%BC%E7%8E%B0%E8%B1%A1" rel="nofollow">龙格现象</a>，解决的办法是分段用较低次数的插值多项式。</p>
<h2><span id=".E9.87.8D.E5.BF.83.E6.8B.89.E6.A0.BC.E6.9C.97.E6.97.A5.E6.8F.92.E5.80.BC.E6.B3.95">重心拉格朗日插值法</span></h2>
<p>重心拉格朗日插值法是拉格朗日插值法的一种改进。在拉格朗日插值法中，运用多项式</p>
<dl><dd><img src="http://upload.wikimedia.org/math/5/0/4/504de244c64091059b93e8d5965b9f02.png" alt="\ell (x)=(x-x_{0})(x-x_{1})\cdots (x-x_{k})" /></dd></dl>
<div>
<div><a class="image" href="http://zh.wikipedia.org/wiki/File:Lagrange_polynomial_Divergence.jpg" rel="nofollow"><img class="thumbimage" src="http://upload.wikimedia.org/wikipedia/commons/thumb/8/8e/Lagrange_polynomial_Divergence.jpg/460px-Lagrange_polynomial_Divergence.jpg" alt="" width="460" height="241" /></a>
<div>
<div><a class="internal" title="放大" href="http://zh.wikipedia.org/wiki/File:Lagrange_polynomial_Divergence.jpg" rel="nofollow"><img src="http://bits.wikimedia.org/static-1.23wmf12/skins/common/images/magnify-clip.png" alt="" width="15" height="11" /></a></div>




拉格朗日插值法的数值稳定性：如图，用于模拟一个十分平稳的函数时，插值多项式的取值可能会突然出现一个大的偏差（图中的14至15中间）</div>




</div>




</div>
<p>可以将拉格朗日基本多项式重新写为：</p>
<dl><dd><img src="http://upload.wikimedia.org/math/f/6/5/f65c55d7b4c330438e7a68482d1590e7.png" alt="\ell _{j}(x)={\frac  {\ell (x)}{x-x_{j}}}{\frac  {1}{\prod _{{i=0,i\neq j}}^{k}(x_{j}-x_{i})}}" /></dd></dl>
<p>定义<strong>重心权</strong><sup id="cite_ref-bary_7-0"><a href="http://zh.wikipedia.org/wiki/%E6%8B%89%E6%A0%BC%E6%9C%97%E6%97%A5%E6%8F%92%E5%80%BC%E6%B3%95#cite_note-bary-7" rel="nofollow">[7]</a></sup><sup id="cite_ref-8"><a href="http://zh.wikipedia.org/wiki/%E6%8B%89%E6%A0%BC%E6%9C%97%E6%97%A5%E6%8F%92%E5%80%BC%E6%B3%95#cite_note-8" rel="nofollow">[8]</a></sup></p>
<dl><dd><img src="http://upload.wikimedia.org/math/f/5/b/f5b96ca21fad59d37cd3ea6a4c70246a.png" alt="w_{j}={\frac  {1}{\prod _{{i=0,i\neq j}}^{k}(x_{j}-x_{i})}}" /></dd></dl>
<p>上面的表达式可以简化为：</p>
<dl><dd><img src="http://upload.wikimedia.org/math/c/f/0/cf0ee67a5ae90253b230f700a86cb5c1.png" alt="\ell _{j}(x)=\ell (x){\frac  {w_{j}}{x-x_{j}}}" /></dd></dl>
<p>于是拉格朗日插值多项式变为：</p>
<dl><dd><img src="http://upload.wikimedia.org/math/e/2/b/e2b06ed5610f4fcc046fef1b0a9f493e.png" alt="L(x)=\ell (x)\sum _{{j=0}}^{k}{\frac  {w_{j}}{x-x_{j}}}y_{j}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ (1)" /></dd></dl>
<p>即所谓的重心拉格朗日插值公式（第一型）或改进拉格朗日插值公式。它的优点是当插值点的个数增加一个时，将每个<img src="http://upload.wikimedia.org/math/e/c/e/ece685c72bdc899bdf745bc4ec290740.png" alt="w_{j}" />都除以<img src="http://upload.wikimedia.org/math/7/e/9/7e9644833883af0bcd5e70adb818536b.png" alt="(x_{j}-x_{{k+1}})" />，就可以得到新的重心权<img src="http://upload.wikimedia.org/math/e/9/2/e926afcf008b0e517216b346c6883710.png" alt="w_{{k+1}}" />，计算复杂度为<img src="http://upload.wikimedia.org/math/d/4/3/d43308deb858cf186f862f2451af08b6.png" alt="{\mathcal  O}(n)" />，比重新计算每个基本多项式所需要的复杂度<img src="http://upload.wikimedia.org/math/5/6/d/56d1e7b3b345d81e79b9ff2a0023adb1.png" alt="{\mathcal  O}(n^{2})" />降了一个量级。</p>
<p>将以上的拉格朗日插值多项式用来对函数<img src="http://upload.wikimedia.org/math/4/a/8/4a866f3e4ce60c1fa64d4bc2100b38da.png" alt="g(x)\equiv 1" />插值，可以得到：</p>
<dl><dd><img src="http://upload.wikimedia.org/math/e/b/3/eb33428c3be74c032a421a8461c184da.png" alt="\forall x,\,g(x)=\ell (x)\sum _{{j=0}}^{k}{\frac  {w_{j}}{x-x_{j}}}" /></dd></dl>
<p>因为<img src="http://upload.wikimedia.org/math/4/a/8/4a866f3e4ce60c1fa64d4bc2100b38da.png" alt="g(x)\equiv 1" />是一个多项式。</p>
<p>因此，将<img src="http://upload.wikimedia.org/math/2/0/5/2054ddef5ff1df5a5e8c1d50a812fefe.png" alt="L(x)" />除以<img src="http://upload.wikimedia.org/math/e/8/4/e84fec1e074026d6fa8e3155482c35c3.png" alt="g(x)" />后可得到：</p>
<dl><dd><img src="http://upload.wikimedia.org/math/0/6/2/0626ddfe1a20a91b8a58862f14efbe83.png" alt="L(x)={\frac  {\sum _{{j=0}}^{k}{\frac  {w_{j}}{x-x_{j}}}y_{j}}{\sum _{{j=0}}^{k}{\frac  {w_{j}}{x-x_{j}}}}}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ (2)" /><sup id="cite_ref-bary_7-1"><a href="http://zh.wikipedia.org/wiki/%E6%8B%89%E6%A0%BC%E6%9C%97%E6%97%A5%E6%8F%92%E5%80%BC%E6%B3%95#cite_note-bary-7" rel="nofollow">[7]</a></sup></dd></dl>
<p>这个公式被称为重心拉格朗日插值公式（第二型）或真正的重心拉格朗日插值公式。它继承了（1）式容易计算的特点，并且在代入<em>x</em>值计算<img src="http://upload.wikimedia.org/math/2/0/5/2054ddef5ff1df5a5e8c1d50a812fefe.png" alt="L(x)" />的时候不必计算多项式<img src="http://upload.wikimedia.org/math/2/5/2/252f274ed0b936c7e5231660fd8bd61f.png" alt="\ell (x)" /><sup id="cite_ref-bary_7-2"><a href="http://zh.wikipedia.org/wiki/%E6%8B%89%E6%A0%BC%E6%9C%97%E6%97%A5%E6%8F%92%E5%80%BC%E6%B3%95#cite_note-bary-7" rel="nofollow">[7]</a></sup>。它的另一个优点是，结合<a title="切比雪夫多项式" href="http://zh.wikipedia.org/wiki/%E5%88%87%E6%AF%94%E9%9B%AA%E5%A4%AB%E5%A4%9A%E9%A1%B9%E5%BC%8F" rel="nofollow">切比雪夫节点</a>进行插值的话，可以很好地模拟给定的函数，使得插值点个数趋于无穷时，最大偏差趋于零<sup id="cite_ref-bary_7-3"><a href="http://zh.wikipedia.org/wiki/%E6%8B%89%E6%A0%BC%E6%9C%97%E6%97%A5%E6%8F%92%E5%80%BC%E6%B3%95#cite_note-bary-7" rel="nofollow">[7]</a></sup>。同时，重心拉格朗日插值结合<a title="切比雪夫多项式" href="http://zh.wikipedia.org/wiki/%E5%88%87%E6%AF%94%E9%9B%AA%E5%A4%AB%E5%A4%9A%E9%A1%B9%E5%BC%8F" rel="nofollow">切比雪夫节点</a>进行插值可以达到极佳的数值稳定性。第一型拉格朗日插值是<a title="数值稳定性" href="http://zh.wikipedia.org/wiki/%E6%95%B0%E5%80%BC%E7%A8%B3%E5%AE%9A%E6%80%A7" rel="nofollow">向后稳定</a>的，而第二型拉格朗日插值是向前稳定的，并且<a class="new" title="勒贝格常数（页面不存在）" href="http://zh.wikipedia.org/w/index.php?title=%E5%8B%92%E8%B4%9D%E6%A0%BC%E5%B8%B8%E6%95%B0&amp;action=edit&amp;redlink=1" rel="nofollow">勒贝格常数</a>很小<sup id="cite_ref-9"><a href="http://zh.wikipedia.org/wiki/%E6%8B%89%E6%A0%BC%E6%9C%97%E6%97%A5%E6%8F%92%E5%80%BC%E6%B3%95#cite_note-9" rel="nofollow">[9]</a></sup>。</p>
